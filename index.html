Deco's cheatsheet...

Справочник для интересностей (чтобы попросту не держать все в голове)

JQuery плагины

Slick-slider

Данная библиотека используется для адаптивной верстки слайдеров. Считается одной из лучших JQuery слайдеров.

Fotorama

Красивый слайдер. Обладает огромным колличеством визуальных настроек, таких как предпросмотр, фуллскрин, и прочие.

Vide.js

Использует видеозаписи в качестве фона для секции, блоков и.т.д Имеет высокую адаптивность под все платформы.

Parallax.js

Используется для создания фона красивым параллакс эффектом для секций сайта. Легко настраивается.

Rellax.js

Параллакс эффекты для отдельных элементов сайта(текста, блоков, графических элементов)

Полезные сайты

Coverr.co

Сток бесплатных видеозаписей для веб сайта, без авторских прав.

Transfonter

Генератор @Font-face стилей для импорта стилей из локальной директории
